Inciante em HTML

  • Na criação de link local onde o arquivo html estiver na mesma pasta basta colocar o nomedoarquivo.html e assim ele irá abrir a pagina.
  • Na criação de link local para irmos para outra pasta devemos colocar nomedapasta/nomedoarquivo.html e assim ele ira para pagina dentro da pasta.
  • Na criação de link local aonde teremos que voltar uma pasta utiliza-se ../nomedoarquivo.html e assim ele retorna a pagina anterior.
  • No html utilizamos a tag vazia < br > para que quebre a linha e pule para a proxima.
  • Utilizamos a tag vazia < img > para incluir imagaem atraves do atributo src" ". A regra dos links para encontrar as imagens é a mesma.
  • No html utilizamos da tag < strong > para deixar uma palavra ou trecho do texto em negrito
  • < input type="text" placeholder="Name" > O comando coloca uma caixa que aceita escrita do tipo texto para nome
  • < input type="text" placeholder="Email"> O comando coloca uma caixa que aceita escrita do tipo texto/caractere para email.
  • A tag < p > cria um paragrafo
  • A tag < em> deixa o texto em italico
  • < input type="range"> Cria uma caixa de controle com barramento
  • < input type="checkbox"> Cria uma caixa pequena de seleção
  • < button> Cria um botão
  • < fildset> Agrupa as entradas relacionadas
  • < form > Cria um formulario para uma tag do tipo < input type=""> e tem que ter uma tag do tipo button para inserir as informações.
  • < legend> colocar a legenda dentro de um formulario

    • Criando Formularios.
    < form >
    < fildset> Username: < input type="text"> < br>
    Password: < input type="password"> < br>
    < button>Login
    < /fildset>


    Nos Formularios rótulos são importantes coloque label antes do nome para garantir a correta coleta de dados exemplo:

    < label> Name < /label> < input type="text"> < br>
    < label> Email < /label> < input type="email"> < br>
    < button> Subscribe < /button>
